SOBRE MIM

Olá! Me chamo Pedro Henrique e estou mergulhando de cabeça no mundo da programação com o objetivo de me tornar um profissional na área. Desde a infância, sempre tive um fascínio por computadores. Passava horas jogando, explorando cada canto dos sistemas operacionais e descobrindo novidades tecnológicas. A paixão por entender como as coisas funcionam por trás da tela sempre me impulsionou a buscar mais conhecimento. Foi então que descobri a programação. Comecei com os conceitos básicos, experimentando linguagens simples como HTML e CSS, e rapidamente me vi encantado pela capacidade de criar algo do zero e ver o resultado ganhar vida na tela do computador.

LINGUAGENS QUE ESTOU APRENDENDO

HTML:

HTML é a estrutura de uma página web. É uma linguagem de marcação que define a estrutura e o conteúdo de uma página da web. Com HTML, você cria elementos como cabeçalhos, parágrafos, listas, links, imagens e muito mais. Cada elemento é marcado com tags, que indicam ao navegador como deve ser exibido. No início achei que era uma linguagem de programação.

CSS:

CSS é uma linguagem de estilização que é usada para poder estilizar uma página web. Com CSS, você pode definir cores, fontes, espaçamento, layout e muito mais. Ele separa o conteúdo da apresentação, permitindo que você altere facilmente o estilo de toda a página ou de elementos individuais.

JAVASCRIPT

JavaScript é uma linguagem de programação que permite adicionar interatividade e funcionalidade a uma página da web. Com JavaScript, você pode responder a eventos do usuário, manipular o conteúdo da página, enviar e receber dados do servidor e muito mais. Por exemplo, você pode usar JavaScript para validar formulários, criar efeitos de animação, carregar conteúdo dinamicamente e criar aplicativos da web completos.
